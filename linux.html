🐧 Linux for Data

Complete Guide

    📁 Directories & Navigation

    Mastering the file tree structure is the first step.

    pwd
    Print Working Directory.
    pwd
    ls
    List directory content.
    ls # Basic list
    ls -l # Long format (permissions, etc.)
    ll # Common alias for ls -l
    ls -la # List all (hidden) + details
    ls -lh # Human readable sizes
    ls -lt # Sort by time
    ls -lS # Sort by size
    ls -r # Reverse order
    ls -lR # Recursive list
    cd
    Change Directory.
    cd /path/to/dir # Go to specific path
    cd .. # Go up one level
    cd ../folder # Go to sibling folder
    cd ~ # Go to home directory
    cd # Go home (shortcut)
    cd ~/folder # Path inside home
    cd - # Back to previous dir
    cd . # Current directory
    mkdir
    Create new directories.
    mkdir new_folder # Create a new directory
    mkdir -p dir1/sub1/sub2 # Create full path (recursive)
    mkdir dir1 dir2 dir3 # Create multiple directories
    rmdir
    Remove empty directories.
    rmdir empty_folder # Remove an empty directory
    rmdir -p path/to/empty # Remove path of empty directories
    tree
    Display directory structure as a tree.
    tree # Display directory structure
    tree -L 2 # Limit display depth to 2 levels
    tree -d # List directories only

    📄 File Operations

    Creating, moving, copying, and deleting files.

    touch
    Create files or update timestamp.
    touch file.txt # Create new file
    touch file1 file2 file3 # Create multiple files
    touch -t 202312251200 file # Set specific time
    cp
    Copy files/folders.
    cp source dest # Copy file
    cp file.txt backup/ # Copy to folder
    cp -r folder1 folder2 # Recursive copy (folder)
    cp -p file backup/ # Preserve permissions
    cp *.txt backup/ # Copy all text files
    mv
    Move or Rename.
    mv oldname newname # Rename
    mv file /path/to/ # Move file
    mv *.log logs/ # Move all log files
    rm
    Remove (Delete).
    rm file.txt # Delete file
    rm -i file.txt # Ask for confirmation
    rm -f file.txt # Force delete
    rm -r folder/ # Recursive delete
    rm -rf folder/ # Force recursive delete
    rm *.tmp # Delete all temp files
    find
    Advanced file search.
    find . -name "*.py" # Search Python files
    find / -size +100M # Files larger than 100MB
    find . -mtime -7 # Modified in last 7 days
    find . -type d -empty # Empty directories
    find . -name "*.log" -delete # Find and delete
    locate
    Quick database search.
    locate filename # Quick search
    updatedb # Update DB (root)
    locate -i name # Case insensitive
    ln
    Create links.
    ln -s target link # Symbolic link
    ln file hardlink # Hard link
    file
    Determine file type.
    file document.pdf # Identify type
    file -b image.jpg # Short output
    file * # Check all files

    📝 Content & Analysis

    Inspect data without opening Excel.

    head / tail
    View start/end.
    head -n 5 data.csv # Check headers
    tail -f app.log # Live monitor
    cat / tac
    Print content.
    cat config.json
    cat parts* > full.csv # Merge files
    grep
    Search inside files.
    grep "ERROR" server.log
    grep -r "db_host" . # Recursive
    wc
    Word Count.
    wc -l data.csv # Count rows

    🔤 Variables & Env

    Managing environment variables is crucial for Data Engineering (e.g., Spark/Hadoop paths).

    export
    Set environment variables.
    export SPARK_HOME=/opt/spark
    echo $SPARK_HOME # Print variable
    env / printenv
    List all variables.
    env | grep PATH
    System Vars
    Built-in variables.
    echo $HOME # Home directory
    echo $USER # Current user

    ⚙️ Pipes & Processing

    Combining commands to create powerful data pipelines.

    Redirection
    Output to files.
    cmd > file.txt # Overwrite
    cmd >> file.txt # Append
    Pipe |
    Pass output to next command.
    cat file.csv | wc -l
    ps aux | grep python
    awk
    Text processing.
    awk -F',' '{print $2}' file.csv
    awk '{s+=$1} END {print s}' nums.txt
    sed
    Stream Editor.
    sed 's/old/new/g' file.txt
    sed -i 's/,/|/g' file.csv # Replace in-place

    ✏️ VI / VIM Editor

    Essential for editing config files on remote servers.

    Modes
    Switching modes.
    vi filename.txt # Open file
    i # Switch to Insert Mode (to type)
    Esc # Switch back to Command Mode
    Save & Quit
    Commands (press Esc first).
    :w # Save
    :q # Quit
    :wq # Save and Quit
    :q! # Quit without saving

    🔐 Permissions

    Control who can Read (r), Write (w), and Execute (x).

    chmod
    Change mode.
    chmod +x script.sh # Make executable
    chmod 755 file # rwx-rx-rx
    chmod 400 key.pem # Protect keys
    chown
    Change owner.
    chown user:group file
    chown -R user folder/ # Recursive

    💻 System & Monitoring

    Keep your servers healthy.

    Resources
    Disk and RAM.
    df -h # Disk space
    du -sh folder/ # Folder size
    free -h # RAM usage
    Processes
    CPU usage.
    top # Real-time monitor
    htop # Visual monitor
    kill 1234 # Stop process ID
    Users
    User management.
    whoami # Current user
    sudo -i # Switch to root
    passwd # Change password

    ⏰ Scheduling (Cron)

    Automate your Python/Bash scripts.

    crontab
    Manage schedule.
    crontab -e # Edit jobs
    crontab -l # List jobs
    0 2 * * * /backup.sh # Run daily at 2AM

    🌐 Networking

    Connecting to servers.

    ssh & scp
    Connect and Transfer.
    ssh user@server
    scp file.csv user@server:/tmp
    Download
    Get files from web.
    wget http://site.com/data.zip
    curl -O http://site.com/data.json

    ⌨️ Keyboard Shortcuts

    Speed up your terminal workflow.

    Keys Action Description
    Ctrl + A Start of line Jump cursor to the beginning
    Ctrl + E End of line Jump cursor to the end
    Ctrl + C Cancel Stop the running command
    Ctrl + D Exit Close terminal or logout
    Ctrl + L Clear Clear the screen
    Ctrl + R History Search Search past commands
    Ctrl + Z Suspend Send process to background
    Ctrl + W Delete Word Delete word backwards
    Tab Autocomplete Auto-complete file/command names
    ↑ / ↓ History Navigate previous commands
