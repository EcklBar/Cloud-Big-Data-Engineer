🐧 Linux for Data

Interactive Guide

📁 Directories & Navigation

Mastering movement within the file tree is the first step.

pwd
Print Working Directory. Where am I?
pwd
ls
List files and directories.
ls -la # Long list + hidden files
ls -lh # Human readable sizes (KB, MB)
ls -lt # Sort by modification time
cd
Change Directory.
cd /var/log # Go to absolute path
cd .. # Go up one level
cd ~ # Go home
cd - # Go to previous directory
mkdir
Make Directory.
mkdir new_folder
mkdir -p data/2024/01 # Create full path

📄 File Operations

Creating, moving, copying, and deleting files.

cp
Copy files or folders.
cp source.csv backup.csv
cp -r data_dir/ backup_dir/ # Recursive copy
mv
Move or Rename.
mv old.txt new.txt # Rename
mv file.csv /tmp/ # Move
rm
Remove (Delete). Careful!
rm file.txt
rm -rf temp_folder # Force recursive delete
find
Search for files in directory structure.
find . -name "*.csv" # Find all CSVs
find /logs -mtime -7 # Modified in last 7 days
find . -size +1G # Files larger than 1GB

📝 Content & Data Analysis

Essential for quick data inspection without opening heavy files.

head / tail
Inspect start or end of files.
head -n 5 data.csv # Check headers
tail -f app.log # Live log monitoring
cat / tac
Output full content.
cat config.json
cat part_* > combined.csv # Merge files
grep
Search text within files (Regex).
grep "ERROR" server.log
grep -r "db_host" . # Recursive search
grep -v "200 OK" access.log # Exclude pattern
wc
Word Count (Lines, Words, Bytes).
wc -l data.csv # Count rows

⚙️ Advanced Processing (Pipes)

The bread and butter of Data Engineering in the shell.

Redirection
Control inputs and outputs.
cmd > output.txt # Overwrite
cmd >> output.txt # Append
cmd 2> error.log # Redirect errors
awk
Powerful text processing language.
awk -F',' '{print $2}' file.csv # Print 2nd column
awk '{s+=$1} END {print s}' nums.txt # Sum column
sed
Stream Editor (Search & Replace).
sed 's/old/new/g' file.txt # Replace text
sed -i 's/,/|/g' data.csv # Replace in-place (CSV to Pipe)
sort & uniq
Sorting and deduplication.
sort names.txt | uniq # Unique list
sort -nr counts.txt # Numeric reverse sort

💻 System & Monitoring

Keep your data pipeline server healthy.

Disk Usage
Check storage space.
df -h # Free disk space
du -sh folder/ # Size of specific folder
du -h --max-depth=1 . # Check sub-folders size
Processes
Monitor CPU and RAM.
top # Real-time process monitor
htop # Better UI monitor
free -h # Check RAM usage
kill
Stop stuck processes.
ps aux | grep python # Find PID
kill 1234 # Terminate PID
kill -9 1234 # Force kill

🌐 Networking & SSH

Connecting to remote servers and transferring data.

ssh
Secure Shell connection.
ssh user@server_ip
ssh -i key.pem user@host
scp
Secure Copy (Transfer files).
scp local_file.csv user@host:/tmp
scp -r user@host:/var/log .
curl / wget
Download data from web.
wget http://example.com/data.zip
curl -O http://example.com/data.json

💡 Data Engineer Pro Tips

1. Quick Data Sampling (Random)

Don't use head for data quality checks, as the top rows might be perfect. Use shuf for a random sample.

shuf -n 5 large_dataset.csv

2. Inspecting Huge CSVs

If you need to check columns but the file is 50GB, don't open it. Just peek.

head -n 5 file.csv # See headers + 4 rows

3. Keyboard Shortcuts

Save time typing commands:

  • Ctrl + R : Search command history.
  • Ctrl + A : Go to start of line.
  • Ctrl + E : Go to end of line.
  • Ctrl + C : Stop running command.
  • Ctrl + L : Clear screen.
