🐋 מהו Docker?

Docker הוא פלטפורמה פופולרית ליצירה, פריסה והרצה של אפליקציות בקונטיינרים

📦 מימוש קונטיינרים

Docker הוא המימוש הפופולרי ביותר של טכנולוגיית הקונטיינרים

הקונטיינרים מאפשרים אריזה של אפליקציות עם כל התלויות שלהן בחבילה אחת ניידת

🛠️ פלטפורמה למפתחים

כלי מתקדם למפתחים ומנהלי מערכות

מאפשר פיתוח, בדיקה ופריסה של אפליקציות בצורה עקבית בכל הסביבות

📋 מדריך מקיף ל-Dockerfile

Dockerfile הוא קובץ מניפסט המתאר איך לבנות Docker Image - המתכון לקונטיינר שלך!

🎯 מטרות Dockerfile

🏗️ יצירת מערכת קבצים

העתקת קבצים מהתיקייה המקומית

⚙️ הכנת הקוד

בנייה והכנה של האפליקציה

📦 הורדת כלים

התקנת חבילות ותוכנות נדרשות

🔗 הגדרת מיפויים

Volumes (מערכת קבצים) ו-Ports (רשת)

📝 פקודות Dockerfile עיקריות

FROM

יבוא image בסיס לשימוש כבסיס

FROM python:3.9-slim FROM ubuntu:22.04 FROM node:16-alpine

ARG

הגדרת פרמטר לבנייה

ARG VERSION=latest ARG BUILD_ENV=production

RUN

הרצת פקודת shell (התקנה, הורדה, יצירה)

RUN apt-get update && apt-get install -y curl RUN pip install -r requirements.txt

WORKDIR

הגדרת תיקיית עבודה בתוך ה-image

WORKDIR /app WORKDIR /workspace

COPY

העתקת קבצים מהתיקייה המקומית

COPY . . COPY requirements.txt /app/

ADD

הוספת קבצים (ממקור מקומי/מרוחק)

ADD https://example.com/file.tar.gz /tmp/ ADD archive.tar.gz /app/

CMD / ENTRYPOINT

פקודת ברירת המחדל להרצה

CMD ["python", "app.py"] ENTRYPOINT ["/app/script.sh"]

EXPOSE

הודעה שהקונטיינר מאזין לפורט

EXPOSE 8080 EXPOSE 3000 5432

VOLUME

יצירת כרך אנונימי

VOLUME ["/data"] VOLUME /var/log

פקודת docker build בונה קונטיינר מתוך Dockerfile

🏷️ דגלי Build עיקריים

# דגל -f לציון קובץ Dockerfile ספציפי docker build -f dockerfile_example . # דגל -t לתיוג ה-image docker build -t image-name:tag . # שימוש בתיקייה הנוכחית docker build .

📝 דוגמאות בנייה

# בנייה עם תג "latest" (ברירת מחדל) docker build -t hello-world . # בנייה עם קובץ Dockerfile אחר docker build -t my-app:from-script -f dockerfile_example2 . # בנייה עם תג מותאם אישית docker build -t name-printer:john .

🚀 הרצת קונטיינרים - Docker Run

🖥️ דגלי Run עיקריים

  • -t: מקצה TTY terminal
  • -i: הרצה אינטראקטיבית
  • --rm: מחיקה אוטומטית בסיום
  • -d: הרצה בפרטי (background)
  • --name: מתן שם לקונטיינר

📋 דוגמאות הרצה

# הרצה פשוטה עם מחיקה אוטומטית docker run --rm -t hello-world # הרצה עם שם מותאם אישית docker run --rm -t --name my-container my-app:from-script # הרצה אינטראקטיבית docker run -it ubuntu bash

💻 תרגילים מעשיים

כאן תוכל להתרגל על יצירת Dockerfile ובנייה של קונטיינרים בצורה מעשית

📝 תרגיל #1: יוצר שמות

🎯 המטלה

  1. כתוב את השם שלך לקובץ טקסט
  2. צור Dockerfile שמשתמש ב-Ubuntu:latest כ-base image
  3. הוסף את הקובץ עם השם לתיקייה "/name-app/my-name.txt" בקונטיינר
  4. הגדר שהקונטיינר יציג את תוכן הקובץ כשהוא רץ (טיפ: פקודת cat)

💡 פתרון מושלם:

# קובץ my-name.txt היי! השם שלי הוא יוסי כהן # Dockerfile FROM ubuntu:latest # יצירת התיקייה RUN mkdir -p /name-app # העתקת קובץ השם COPY my-name.txt /name-app/ # הצגת התוכן כשהקונטיינר רץ CMD ["cat", "/name-app/my-name.txt"]

🔨 פקודות בנייה והרצה:

# בנייה עם השם שלך כתג docker build -t name-printer:yossi . # הרצה docker run --rm name-printer:yossi

🌟 תרגיל #2: שרת Python פשוט

🎯 המטלה המתקדמת

  1. צור קובץ Python פשוט שמציג "Hello from Docker!"
  2. צור Dockerfile עם Python:3.9-slim כבסיס
  3. העתק את קובץ ה-Python לקונטיינר
  4. הגדר את הקונטיינר להריץ את הסקריפט

💡 פתרון מקצועי:

# קובץ app.py #!/usr/bin/env python3 print("🐋 Hello from Docker!") print("Python version:", __import__('sys').version) # Dockerfile FROM python:3.9-slim WORKDIR /app COPY app.py . CMD ["python", "app.py"]

🔨 פקודות בנייה והרצה:

# בנייה docker build -t python-hello . # הרצה docker run --rm python-hello

🏆 תרגיל #3: אפליקציית נתונים

🎯 המטלה למהנדסי נתונים

  1. צור קובץ Python שקורא CSV ומציג סטטיסטיקות בסיסיות
  2. צור קובץ requirements.txt עם pandas
  3. צור Dockerfile שמתקין את התלויות
  4. הרץ ניתוח על קובץ נתונים דמה

💡 פתרון מקצועי למהנדסי נתונים:

# requirements.txt pandas==1.5.3 numpy==1.24.3 # data_analyzer.py import pandas as pd import numpy as np # יצירת נתונים לדוגמה data = { 'name': ['Alice', 'Bob', 'Charlie', 'Diana'], 'age': [25, 30, 35, 28], 'salary': [50000, 60000, 70000, 55000] } df = pd.DataFrame(data) print("🔍 ניתוח נתונים עם Docker:") print("="*40) print("📊 הנתונים:") print(df) print("\n📈 סטטיסטיקות:") print(f"ממוצע גיל: {df['age'].mean():.1f}") print(f"ממוצע שכר: ${df['salary'].mean():,.0f}") print(f"מספר רשומות: {len(df)}") # Dockerfile FROM python:3.9-slim WORKDIR /app COPY requirements.txt . RUN pip install -r requirements.txt COPY data_analyzer.py . CMD ["python", "data_analyzer.py"]

🔨 פקודות בנייה והרצה:

# בנייה docker build -t data-analyzer . # הרצה docker run --rm data-analyzer

🔗 רשתות Docker

Bridge Network

רשת ברירת המחדל לקונטיינרים על אותו Host

docker network create my-bridge docker run --network my-bridge nginx

Host Network

שימוש ישיר ברשת של ה-Host

docker run --network host nginx

None Network

בידוד מוחלט ללא גישה לרשת

docker run --network none alpine

💾 Docker Volumes

Volumes מאפשרים שמירת נתונים מחוץ לקונטיינר ושיתוף נתונים בין קונטיינרים

Named Volume

docker volume create my-data docker run -v my-data:/data nginx

Bind Mount

docker run -v /host/path:/container/path nginx

Anonymous Volume

docker run -v /data nginx
