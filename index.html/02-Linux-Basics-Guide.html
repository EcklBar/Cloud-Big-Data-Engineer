הפצות Linux

מהי הפצת Linux?

  • אוסף של תוכנות (בדרך כלל קוד פתוח) על גבי ליבת Linux
  • כוללת תוכנות שרת, כלי ניהול מערכת, תיעוד ויישומי שולחן עבודה
  • מאוחסנת במאגר תוכנה מרכזי ומאובטח
  • מספקת מראה ותחושה אחידים, ניהול תוכנה קל ומטרה תפעולית ספציפית

הפצות מובילות

Red Hat

  • חברת Linux מסחרית בשווי מיליארד דולר
  • מאות מומחי Linux ותמיכה מעולה
  • מוצרים חינמיים: Red Hat Enterprise Linux ו-Fedora

Ubuntu

  • החלה לשלוח דיסקים חינמיים ב-2004
  • פופולרית מאוד למשתמשים ביתיים
  • המטרה: שולחן עבודה גרפי ללא צורך בשורת פקודה

Debian

  • אלפי מפתחים מאורגנים בוחרים מנהיג פרויקט כל שנתיים
  • נחשבת לאחת ההפצות היציבות ביותר
  • גרסאות נקראות על שם דמויות מהסרט Toy Story

CentOS

  • פלטפורמת מחשוב חינמית ברמה ארגונית
  • נתמכת על ידי הקהילה
  • ב-2014 הצטרפה ל-RHEL תוך שמירה על עצמאות

הפצות נוספות

  • Oracle Enterprise Linux ו-Scientific Linux - מבוססות על Red Hat Enterprise Linux
  • Linux Mint, Edubuntu ועוד רבות - מבוססות על Ubuntu
  • בסך הכל קיימות מאות הפצות Linux שונות!

למה להשתמש בכלים המודרניים?

כלים אלו מאיצים משמעותית את העבודה היומיומית, מקלים על ניווט וניהול שרתים, ומספקים חוויית משתמש מעולה עם יכולות אוטומציה מתקדמות.

התחברות באמצעות PuTTY

שלבי ההתחברות:

  1. הורדת PuTTY מ-www.putty.org
  2. יצירת חיבור חדש
  3. הזנת כתובת IP של שרת Linux ופורט SSH (22 כברירת מחדל)
  4. מתן שם ושמירת ה-Session
  5. בפתיחת ה-Session - הזנת שם משתמש וסיסמה
נאי/naya (עבור DE's machine)

פתרון בעיות:

  • וידוא שהשרת פועל
  • בדיקת כתובת IP ומספרי פורט
  • וידוא שה-Firewall לא חוסם חיבורים

קונספט מרכזי: Everything is a File

העיקרון המרכזי של Unix/Linux

"הכל הוא קובץ" - גישה לעיצוב ממשק במערכות Unix ונגזרותיה

מה זה אומר בפועל?

📄 קבצים רגילים

  • מסמכים, תמונות, סרטונים
  • קבצי תצורה וסקריפטים
  • קבצים בינאריים והפעלה

📁 תיקיות

  • גם תיקיות הן "קבצים מיוחדים"
  • מכילות רשימה של קבצים אחרים
  • ניתן לקרוא אותן כמו קובץ

🔧 התקני חומרה

  • /dev/sda - דיסק קשיח
  • /dev/tty - טרמינל
  • /dev/null - "חור שחור"
  • /dev/random - מספרים אקראיים

היתרונות של הגישה

  • אחידות: אותן פקודות עובדות על הכל - cat, echo, grep
  • פשטות: אין צורך בממשקים מיוחדים לכל סוג משאב
  • גמישות: ניתן להפנות פלט מכל דבר לכל דבר
  • כוח: שליטה מלאה במערכת דרך ממשק אחיד

דוגמאות מעשיות

# קריאת מידע על המעבד
cat /proc/cpuinfo

# שליחת הודעה לטרמינל אחר
echo "Hello" > /dev/pts/1

# מחיקת נתונים מיותרים
command_with_output > /dev/null

# קריאת נתונים אקראיים
head -c 100 /dev/random | base64

תרגיל 1 - עבודה עם תיקיות (Directories)

  1. הצג את התיקיה הנוכחית שלך
    pwd
  2. עבור לתיקיה /etc
    cd /etc
  3. עבור לתיקיית הבית שלך בשלושה לחיצות מקשים בלבד
    cd ~
  4. עבור לתיקיה /var/log באמצעות מקש Tab
    cd /var/log [TAB]
  5. הצג את תוכן התיקיה /var/log
    ls /var/log
  6. עבור לתיקיית האב של התיקיה הנוכחית
    cd ..
  7. הישאר היכן שאתה והצג את תוכן /etc
    ls /etc
  8. הישאר היכן שאתה והצג את תוכן /bin ו-/sbin
    ls /bin /sbin
  9. הישאר היכן שאתה והצג את תוכן תיקיית הבית (~)
    ls ~
  10. הצג את כל הקבצים (כולל נסתרים) בתיקיית הבית שלך
    ls -la ~
  11. הצג את הקבצים ב-/boot בפורמט קריא לאדם (גודל)
    ls -lh /boot
  12. צור תיקיה בשם testdir בתיקיית הבית שלך
    mkdir ~/testdir
  13. עבור ל-/etc, הישאר שם וצור תיקיה בשם newdir בתיקיית הבית
    cd /etc
    mkdir ~/newdir
  14. צור את התיקיות הבאות עם תת-תיקיות:
    mkdir -p ~/tutorial/linux/dir1/dir2/dir3
    • dir3 היא תת-תיקיה של dir2
    • dir2 היא תת-תיקיה של dir1
  15. מחק את התיקיה testdir
    rmdir ~/testdir

תרגיל 2 - עבודה עם קבצים (Files)

  1. הצג את הקבצים בתיקיה /bin/
    ls /bin/
  2. צור קובץ בשם "myfile.txt" בתיקיית הבית שלך
    touch ~/myfile.txt
  3. שנה את שם הקובץ ל-"mynewfile.txt"
    mv ~/myfile.txt ~/mynewfile.txt
  4. העתק את הקובץ ל-"mycopy.txt"
    cp ~/mynewfile.txt ~/mycopy.txt
  5. צור תיקיה בשם "tmpdir" בתיקיית הבית שלך
    mkdir ~/tmpdir
  6. העתק את הקובץ שלך לתיקיה זו
    cp ~/mycopy.txt ~/tmpdir/
  7. הצג את תוכן התיקיה וודא שהקובץ הועתק
    ls ~/tmpdir/
  8. מחק את התיקיה ואת תוכנה בפקודה אחת
    rm -rf ~/tmpdir
  9. מחק את כל הקבצים שיצרת בתרגיל זה
    rm ~/mynewfile.txt ~/mycopy.txt

תרגיל 3 - עבודה עם תוכן קבצים

  1. הצג את 12 השורות הראשונות של /etc/services
    head -12 /etc/services
  2. הצג את השורה האחרונה של /etc/passwd
    tail -1 /etc/passwd
  3. השתמש ב-cat כדי ליצור קובץ בשם count.txt בתיקיית הבית עם השורות הבאות:
    cat > ~/count.txt
    One
    Two
    Three
    Four
    Five
    [Ctrl+D]
  4. השתמש ב-cat כדי ליצור גיבוי של הקובץ ל-catBCK.txt
    cat ~/count.txt > ~/catBCK.txt
  5. השתמש גם ב-"more" וגם ב-"less" להצגת תוכן /etc/services
    more /etc/services
    less /etc/services

    נסה לראות את ההבדל בין שתי הפקודות

תרגיל 4 - עבודה עם משתנים (Variables)

  1. השתמש ב-echo להצגת Hello ואחריו שם המשתמש שלך (משתנה bash)
    echo Hello $USER
  2. צור משתנה בשם answer עם הערך 42
    answer=42
  3. העתק את הערך של $LANG ל-$MyLANG והצג אותו
    MyLANG=$LANG
    echo $MyLANG
  4. בטל את המשתנה answer
    unset answer
  5. צור משתנה והקצה לו את הערך 'Hello', צור משתנה נוסף עם הערך 'World', השתמש ב-echo ובשני המשתנים כדי להציג 'HelloWorld'
    var1='Hello'
    var2='World'
    echo $var1$var2
  6. ייצא את המשתנה שלך
    export var1

תרגיל 5 - I/O Redirection

  1. הפנה את הפלט של pwd עם המחרוזת 'Files Under path:' והדפס את כל הקבצים תחת תיקיית הבית שלך לקובץ (צור קבצים בתיקייה אם נדרש)
    echo "Files Under path: $(pwd)" > ~/file_list.txt
    ls ~ >> ~/file_list.txt

    דוגמה לפלט:

    Files Under path: /home/naya

    linux1.txt
    linux.txt
    part1
    part2
    part3
    system.log

תרגיל 6 - Pipes ו-Filters

  1. השתמש ב-pipe על הקובץ /var/log/messages:
    • ספור את מספר השורות
      cat /var/log/messages | wc -l
    • הצג את 4 השורות הראשונות מ-10 השורות האחרונות
      tail -10 /var/log/messages | head -4
  2. השתמש ב-grep על הקובץ /var/log/messages:
    • הצג רק שורות מהחודש הנוכחי
      grep "$(date +%b)" /var/log/messages
    • הצג שורות תוך ביטול כל השורות מהחודש הנוכחי
      grep -v "$(date +%b)" /var/log/messages
    • הצג שורות המכילות את המילה "ERROR", תוך התעלמות מגודל אותיות
      grep -i "ERROR" /var/log/messages
  3. צור קובץ חדש (sort.file) עם התוכן הבא:
    cat > ~/sort.file
    C3
    A1
    B2
    D4
    [Ctrl+D]
    • הצג את תוכן הקובץ
      cat ~/sort.file
    • מיין את תוכן הקובץ
      sort ~/sort.file
    • מיין את תוכן הקובץ בסדר הפוך
      sort -r ~/sort.file
    • הצג רק את הערך "C3" (בשתי שיטות)
      grep "C3" ~/sort.file
      grep -w "C3" ~/sort.file
    • הצג רק את הערך "B2" (בשתי שיטות)
      grep "B2" ~/sort.file
      sed -n '/B2/p' ~/sort.file
  4. צור קובץ חדש (uniq.file) עם התוכן הבא:
    cat > ~/uniq.file
    C3
    A1
    A1
    B2
    D4
    A1
    [Ctrl+D]
    • הצג את תוכן הקובץ
      cat ~/uniq.file
    • הסר ערכים כפולים
      sort ~/uniq.file | uniq
    • הסר כפילויות ומיין את הפלט בסדר הפוך
      sort ~/uniq.file | uniq | sort -r

תרגיל 7 - שימוש בעורך VI

  1. הפעל את כלי vi ובצע את הפעולות הבאות (שם קובץ: file.txt):
    vi ~/file.txt
  2. הכנס: "This is the first line"
    i
    This is the first line
    [ESC]
  3. הכנס: "This is the second line"
    o
    This is the second line
    [ESC]
  4. הכנס: This line is going to be deleted
    o
    This line is going to be deleted
    [ESC]
  5. מחק את השורה השלישית באמצעות פקודת "delete row"
    dd
  6. שנה את המילה "first" למילה "edited"
    /first
    cw
    edited
    [ESC]
  7. שמור את הקובץ וצא בפקודה אחת
    :wq
  8. הצג את הקובץ שלך באמצעות "cat"
    cat ~/file.txt

תרגיל 8 - הרשאות קבצים (File Permissions)

  1. צור קובץ חדש בשם "mysecret.txt"
    touch ~/mysecret.txt
  2. בדוק את הרשאות הקובץ הנוכחיות ברמות "owner", "group" ו-"other"
    ls -l ~/mysecret.txt
  3. הוסף הרשאת "Write" ברמות "group" ו-"other"
    chmod g+w,o+w ~/mysecret.txt
  4. באמצעות סימון נומרי, הוסף הרשאות מלאות ברמת owner ו-group, ו-"Read" ו-"Execute" ברמת "other"
    chmod 775 ~/mysecret.txt
  5. אמת את הרשאות הקובץ שלך
    ls -l ~/mysecret.txt
