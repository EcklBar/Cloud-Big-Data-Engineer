Topic Management

Create a Topic

Create a new topic with specific partitions and replication factor.

kafka-topics.sh --create --topic my-topic --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1

List all Topics

View a list of all available topics on the cluster.

kafka-topics.sh --list --bootstrap-server localhost:9092

Describe a Topic

View details like partition count, leader, and replicas.

kafka-topics.sh --describe --topic my-topic --bootstrap-server localhost:9092

Delete a Topic

Permanently remove a topic (marks for deletion).

kafka-topics.sh --delete --topic my-topic --bootstrap-server localhost:9092

Producers and Consumers

Start a Console Producer

Read input from standard in and write to Kafka.

kafka-console-producer.sh --broker-list localhost:9092 --topic my-topic

Start a Console Consumer

Read data from the topic and print to standard out (from beginning).

kafka-console-consumer.sh --bootstrap-server localhost:9092 --topic my-topic --from-beginning

Consumer Group Management

Describe Consumer Group

Check the lag and offset positions for a specific group.

kafka-consumer-groups.sh --bootstrap-server localhost:9092 --group my-group --describe

Advanced & Reference

Configuration Management

kafka-configs.sh: Manage configurations for topics, brokers, and clients.

kafka-configs.sh --bootstrap-server localhost:9092 --entity-type topics --entity-name my-topic --describe

Reassign Partitions

kafka-reassign-partitions.sh: Move partition replicas between brokers.

Log Directories

kafka-log-dirs.sh: Describe log directories on the brokers.

kafka-log-dirs.sh --bootstrap-server localhost:9092 --describe

Delete Records

kafka-delete-records.sh: Delete records from a topic before the retention period ends.

Access Control Lists (ACLs)

kafka-acls.sh: Manage Access Control Lists for securing your cluster.

Performance Verification

kafka-verifiable-producer.sh & kafka-verifiable-consumer.sh: Tools to test and verify throughput and performance.
