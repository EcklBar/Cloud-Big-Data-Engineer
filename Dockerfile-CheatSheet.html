🧾 Dockerfile Cheat Sheet PRO EDITION

Production-grade best practices, security hardening, and optimization techniques.

🏆 1. The Production Template

Secure, small, and fast. Includes non-root user and explicit versioning. 

# 1. Pin specific version (Avoid 'latest')
FROM python:3.10.4-slim-bullseye

# 2. Set envs to prevent python buffering
ENV PYTHONDONTWRITEBYTECODE=1 \
    PYTHONUNBUFFERED=1

WORKDIR /app

# 3. Create non-root user for security
RUN useradd -m appuser

# 4. Install Dependencies
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# 5. Copy code & switch user
COPY . .
USER appuser

# 6. Use Exec form (JSON array)
CMD ["python", "main.py"]

🧹 2. The "Apt-Get" Cleanup

Reduce image size by ~30MB+

  • Combine update, install, and clean in ONE layer.
  • If you split them, the cache remains in the history.
RUN apt-get update && apt-get install -y \
    gcc \
    libpq-dev \
    # Clean up in the same layer!
    && rm -rf /var/lib/apt/lists/*

🛑 3. Signal Handling (PID 1)

Ensure `Ctrl+C` or Kubernetes kills work.

BAD (Shell Form) 
CMD python main.py
Starts as /bin/sh -c. Signals get swallowed.
GOOD (Exec Form) 
CMD ["python", "main.py"]
Runs directly. Signals pass through.

📚 4. Intelligent Caching

Docker caches lines from top to bottom.

  • ❌ Copy All First: Any code change invalidates dependency install.
  • ✅ Copy Reqs First: Dependencies are cached unless requirements.txt changes.
COPY requirements.txt .
RUN pip install ... # Cached!
COPY . .            # Changes often

🔐 5. Secrets Management

NEVER use ENV for passwords/keys. They persist in docker history.

The Fix (BuildKit): 
# Mount secret only for this command
RUN --mount=type=secret,id=mykey \
    cat /run/secrets/mykey | scripts/install.sh

🏗️ 6. The "Shrinker" (Multi-Stage)

Build with tools (gcc, go), ship only the binary.

FROM golang:1.18 as builder
RUN go build -o app main.go

# Final image is tiny
FROM alpine:3.14
COPY --from=builder /app /app
CMD ["/app"]

❤️ 7. Add Healthchecks

Let Docker restart your app if it freezes.

HEALTHCHECK --interval=30s --timeout=3s \
  CMD curl -f http://localhost:8000/ || exit 1

🏷️ 8. Version Pinning

  • node:latest
    Unpredictable. Might break tomorrow.
  • node:20.1-alpine
    Stable. Predictable size and behavior.
