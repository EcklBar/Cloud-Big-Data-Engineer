Variables and Data Types

Python is dynamically typed, meaning you don't need to declare variable types explicitly. Let's explore the fundamental data types and how to work with them.

6. üìö Working with Dictionaries

Dictionaries are key-value pairs where keys must be unique and are not indexed by position:

# List comprehensions numbers = [1, 2, 3, 4, 5] squares = [x**2 for x in numbers] print(f"Squares: {squares}") # Even numbers only evens = [x for x in numbers if x % 2 == 0] print(f"Even numbers: {evens}") # Dictionary comprehension word_lengths = {word: len(word) for word in ['python', 'data', 'science']} print(f"Word lengths: {word_lengths}") # Functions def greet(name, greeting="Hello"): return f"{greeting}, {name}!" print(greet("Alice")) print(greet("Bob", "Hi")) # Lambda functions multiply = lambda x, y: x * y print(f"5 * 3 = {multiply(5, 3)}") # Map and filter doubled = list(map(lambda x: x * 2, numbers)) print(f"Doubled: {doubled}") filtered = list(filter(lambda x: x > 2, numbers)) print(f"Greater than 2: {filtered}")
