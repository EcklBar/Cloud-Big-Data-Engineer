    üìÅ Directories & Navigation

    Mastering the file tree structure is the first step.

    pwd
    Print Working Directory.
    pwd
    ls
    List directory content.
    ls # Basic list
    ls -l # Long format (permissions, etc.)
    ll # Common alias for ls -l
    ls -la # List all (hidden) + details
    ls -lh # Human readable sizes
    ls -lt # Sort by time
    ls -lS # Sort by size
    ls -r # Reverse order
    ls -lR # Recursive list
    cd
    Change Directory.
    cd /path/to/dir # Go to specific path
    cd .. # Go up one level
    cd ../folder # Go to sibling folder
    cd ~ # Go to home directory
    cd # Go home (shortcut)
    cd ~/folder # Path inside home
    cd - # Back to previous dir
    cd . # Current directory
    mkdir
    Create new directories.
    mkdir new_folder # Create a new directory
    mkdir -p dir1/sub1/sub2 # Create full path (recursive)
    mkdir dir1 dir2 dir3 # Create multiple directories
    rmdir
    Remove empty directories.
    rmdir empty_folder # Remove an empty directory
    rmdir -p path/to/empty # Remove path of empty directories
    tree
    Display directory structure as a tree.
    tree # Display directory structure
    tree -L 2 # Limit display depth to 2 levels
    tree -d # List directories only

    üìÑ File Operations

    Creating, moving, copying, and deleting files.

    touch
    Create files or update timestamp.
    touch file.txt # Create new file
    touch file1 file2 file3 # Create multiple files
    touch -t 202312251200 file # Set specific time
    cp
    Copy files/folders.
    cp source dest # Copy file
    cp file.txt backup/ # Copy to folder
    cp -r folder1 folder2 # Recursive copy (folder)
    cp -p file backup/ # Preserve permissions
    cp *.txt backup/ # Copy all text files
    mv
    Move or Rename.
    mv oldname newname # Rename
    mv file /path/to/ # Move file
    mv *.log logs/ # Move all log files
    rm
    Remove (Delete).
    rm file.txt # Delete file
    rm -i file.txt # Ask for confirmation
    rm -f file.txt # Force delete
    rm -r folder/ # Recursive delete
    rm -rf folder/ # Force recursive delete
    rm *.tmp # Delete all temp files
    find
    Advanced file search.
    find . -name "*.py" # Search Python files
    find / -size +100M # Files larger than 100MB
    find . -mtime -7 # Modified in last 7 days
    find . -type d -empty # Empty directories
    find . -name "*.log" -delete # Find and delete
    locate
    Quick database search.
    locate filename # Quick search
    updatedb # Update DB (root)
    locate -i name # Case insensitive
    ln
    Create links.
    ln -s target link # Symbolic link
    ln file hardlink # Hard link
    file
    Determine file type.
    file document.pdf # Identify type
    file -b image.jpg # Short output
    file * # Check all files

    üìù Content & Analysis

    Inspect data without opening Excel.

    cat
    Print content.
    cat file.txt
    cat -n file.txt # Line numbers
    cat f1 f2 > merged
    cat >> file.txt # Append
    cat > file.txt # Create file (Ctrl+C to stop)
    tac
    Reverse cat (Bottom-up).
    tac file.txt
    echo -e "1\n2" | tac
    less / more
    Paged view.
    less file.txt # Scrollable
    more file.txt # Forward only
    Keys: q=quit, /search, n=next
    head / tail
    Start / End of file.
    head -n 20 file.txt
    tail -n 50 file.txt
    tail -f log.txt # Live monitor
    grep
    Search text.
    grep "pattern" file
    grep -i "pat" file # No-case
    grep -r "pat" dir/ # Recursive
    grep -v "pat" file # Invert match
    grep -c "pat" file # Count
    ripgrep (rg)
    Modern, faster grep.
    rg "pattern"
    rg -n "pat" # Line nums
    rg -t py "import" # Py files only
    rg --hidden "secret"
    diff
    Compare files.
    diff file1 file2
    diff -y file1 file2 # Side-by-side
    diff -r dir1 dir2

    üî§ Variables & Shell

    Environment variables, bash syntax, and quoting.

    Definitions
    Set variables.
    VAR="value"

    readonly VAR="val"
    unset VAR
    echo
    Print output.
    echo $VAR

    echo 'Hi $USER' # Literal
    echo -e "A\nB" # Newline
    env
    Environment info.

    printenv PATH
    env | grep HOME
    System Vars
    Built-ins.
    $HOME, $USER, $PATH
    $PWD, $SHELL, $HOSTNAME
    Command Sub $()
    Run cmd inside cmd.
    x=$(cat file.txt)
    today=$(date +%F)
    Scoping {}
    Delimit variables.
    echo "${VAR}man"
    ${VAR:-default} # Default val
    Wildcards
    Pattern matching.
    *.txt # All .txt
    file?.txt # file1, fileA
    [a-z]* # Lowercase start
    Specials # \
    Comments & Escaping.
    # This is a comment
    echo \$100 # Literal $
    echo line \ # Continue line

    üîÑ I/O & Redirection

    Manage Standard Input, Output, and Error.

    Output >
    Redirect stdout.
    cmd > file.txt # Overwrite
    cmd >> file.txt # Append
    cmd > /dev/null # Discard
    Input <
    Redirect stdin.
    cmd < input.txt
    cmd << EOF # Here Doc
    cmd <<< "str" # Here String
    Error 2>
    Redirect stderr.
    cmd 2> err.log
    cmd > out 2>&1 # Both to same
    cmd &> all.log # Short syntax
    tee
    Split output (Screen+File).
    cmd | tee log.txt
    cmd | tee -a log.txt # Append

    ‚öôÔ∏è Pipes & Processing

    Advanced text manipulation pipelines.

    Pipe |
    Chain commands.

    cut
    Slice columns.

    Sorting.
    sort -n file # Numeric
    sort -r file # Reverse
    uniq -c # Count uniques
    sort | uniq
