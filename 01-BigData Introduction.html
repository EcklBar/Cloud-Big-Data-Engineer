יתרונות מחשוב ענן

💰 עלות (Cost)

לא צריך יותר:

  • לקנות חומרה ותוכנה
  • להקים ולתפעל מרכזי נתונים באתר

⚡ מהירות (Speed)

  • Self service - קבלת משאבי מחשוב עצומים תוך דקות
  • מסיר את האחריות לתכנון קיבולת

🌍 סקאלה גלובלית

  • יכולת לגדול באופן אלסטי
  • הוספה/הפחתה של כוח מחשוב, אחסון ורוחב פס לפי הצורך

🚀 פרודוקטיביות

צוותי IT יכולים להקדיש זמן להשגת יעדים עסקיים חשובים יותר

📊 ביצועים

  • רשת עולמית של מרכזי נתונים מאובטחים
  • שדרוג מתמיד לחומרה החדשה ביותר
  • טוב יותר ממרכזי נתונים בודדים בארגון

🔒 אבטחה

  • סט של מדיניות, טכנולוגיות ובקרות שמחזקות את תנוחת האבטחה
  • הגנה על הנתונים, האפליקציות והתשתית מפני איומים פוטנציאליים

Batch vs Streaming Processing

📦 Batch Processing

עיבוד קבוצתי:

  • מטפל בסטים גדולים של נתונים במרווחי זמן מוגדרים
  • תהליך ETL מסורתי - העברת נתונים ממקורות ליעדים
  • יעיל כאשר לא נדרשת זמינות נתונים בזמן אמת

⚡ Streaming Processing

עיבוד בזמן אמת:

  • נתונים נוצרים ומועברים ברציפות ממקורות רבים
  • אלפי קלטי נתונים בגדלים קטנים (KB) במקביל
  • איסוף נתונים ממכשירים מפוזרים גיאוגרפית בזמן יצירתם

הולדת הביג דאטה

האתגר

מערכות RDBMS מסורתיות לא יכלו להתמודד עם סקאלות של ביג דאטה

  • חברות נאלצו למצוא פתרונות חלופיים
  • BigData ו-NoSQL נכנסו לעולם והציגו מגוון רחב של פתרונות נתונים
  • יכולות חדשות: רכישה, אחסון וניתוח של נפחי נתונים עצומים

הענן הציבורי מציע סט מקיף ויעיל של שירותים מנוהלים לרכישה, אחסון וניתוח נתונים בכל סקאלה ומהירות.

ACID vs BASE

🔒 ACID

Synchronous Replication

  • רפליקציה סינכרונית בין Master ו-Replica
  • המתנה לאישור כתיבה משני הצדדים
  • עקביות מלאה של הנתונים

🔄 BASE

Asynchronous Replication

  • רפליקציה א-סינכרונית
  • אישור מיידי ללקוח לאחר כתיבה ל-Master
  • עקביות בסופו של דבר (Eventual Consistency)

Apache Kafka

🔄 פלטפורמת ה-Streaming המובילה

משמשת כ"מרכז מבוזר" בו Producers מפרסמים הודעות לנושאים (topics) ספציפיים, ו-Consumers נרשמים לנושאים אלה כדי לקבל את ההודעות בזמן אמת.

ארכיטקטורה:

  • Producers - שולחים הודעות ל-Kafka Cluster
  • Topics - מחולקים ל-Partitions לסקלביליות
  • Consumers - קוראים הודעות מה-Topics

שימושים נפוצים:

Web Apps Microservices Monitoring Analytics IoT CRMs Databases Data Lakes
